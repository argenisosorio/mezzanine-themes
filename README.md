# Themes for The Best Django CMS, Mezzanine

## Versiones
```
Mezzanine==4.2.2
Django==1.10.6
```

## Themes
- tema_generico (Default theme for mezzanine)

## Installation
```
# pip install -r requirements.txt

$ python manage.py migrate

$ python manage.py createsuperuser

$ python manage.py migrate

$ python manage.py runserver
```
